# Overview Monorepo

It is an open source project that provides all the front-end interfaces of LaWallet. <br/>
Includes the front-end interface of the web application, react hooks library and atomic LaWallet components. <br/>
This repository uses <Link className="text-blue-500" href="https://pnpm.io/workspaces">pnpm workspaces</Link> to manage multiple projects.
